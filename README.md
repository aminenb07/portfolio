# Antique Journal Portfolio

An immersive, "museum-grade" digital portfolio experience designed to evoke the feeling of reading an old, physical manuscript.

## 📖 The Vision
This project departs from modern "glossy" UI trends, instead focusing on:
- **Paper & Ink**: Aged parchment textures and deep ink-black typography.
- **Physical Interaction**: A bespoke page-turning system that mimics the weight and friction of heavy paper.
- **Tactile Audio**: Real-time synthesized soundscapes for scrolling, selecting text, and flipping pages.

## 🛠️ Tech Stack
- **Next.js**: Framework for performance and semantic structure.
- **GSAP & ScrollTrigger**: Powering the complex physical animations and page transitions.
- **Web Audio API**: Real-time synthesis of tactile sounds (no external audio assets).
- **Vanilla CSS**: Custom styling engine for the parchment aesthetic.

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **View the Journal**:
   Open [http://localhost:3000](http://localhost:3000) (unmute to experience the full atmosphere).
